## Environments variables

```
# Discord :
BOT_TOKEN="your token"
CLIENT_ID="bot id"
WEBHOOK_URL="a webhook url"

# Database:
DATABASE_URL=""

# OpenAI :
GITHUB_TOKEN="your openai key"

# Stripe:
STRIPE_PUBLIC_KEY=""
STRIPE_PRIVATE_KEY=""
STRIPE_LIVE_MODE=""

# Miscs:
DISCORD_SUPPORT_URL=""
SUPPORT_GUILD_ID=""
SUPPORT_PREMIUM_ROLE_ID=""
OWNER_ID=""
OWNERS=""
STRIPE_YEARLY_SUB=""
STRIPE_MONTHLY_SUB=""
STRIPE_DASHBOARD_URL=""
```
